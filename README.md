## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sentiment_tracker'
```

And then execute:

```bash
$ bundle install
```

Or install it yourself as:

```bash
$ gem install sentiment_tracker
```

Ones installed, generate migrations:

```bash
$ rails generate sentiment_tracker
```

Then:

```bash
$ rails db:migrate
```
